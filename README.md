🚀 FocusHub - Productivity Dashboard

A modern and responsive Productivity Dashboard built using HTML, CSS, and JavaScript. FocusHub helps users stay organized with task management, daily planning, Pomodoro sessions, weather updates, motivational quotes, daily goals, and a smart digital clock — all in one clean interface.
